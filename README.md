# Resume Analyzer

A web application that analyzes a resume against a job description and provides a match score along with missing keywords.

## 🚀 Features

* Upload PDF resume
* Compare with job description
* Match percentage calculation
* Highlights missing skills
* Smart filtering (removes duplicate & common words)

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* PDF.js

## 🌐 Live Demo

https://resume-analyzer-charan.netlify.app/

## 📌 How it works

1. Extracts text from resume PDF
2. Processes and filters keywords
3. Compares with job description
4. Displays match score and suggestions
